Project Details

SpringBoard

This Basecamp clone allows multiple todo-lists and a rich-text-formatted message board in one easy to navigate dashboard.

  • Utilized React-Redux containers to architect modular components which were reused for 75% of the front end keeping the codebase DRY.
  • Integrated the React-Quill library which allows the user to create rich-text notes without any change to backend configuration or increase in overhead.
  • Optimized back end routes using Rails and JBuilder in order to decrease queries giving the user less load-time for more information.

plantr

Collect your houseplants in a 'garden' to receive watering and check in reminders.

  • Utilized mongoose schema methods to create a joins architecture giving users the ability to collect plants in their digital garden.
  • Created validations for MongoDB documents taking advantage of mongoose validator methods.
  • Implemented User Auth by encrypting user details via the JSON Webtoken library to be sent over HTTP ensuring unencrypted passwords are not saved to our database.

CSA Coordinator

(in progress)

My local Community Supported Agriculture organization currently collects data and tracks volunteer attendance manually. PLGCSA Coordinator will simplify record keeping and help coordinators contact volunteers.

  • Utilizes SignUpGenius's external API to populate volunteer rosters and limits requests to keep the cost down for the non-profit.
  • Using Ruby on Rails through associations, tracks the progress of a share's (made up of many volunteers) progress toward membership requirements.
  • Individual React components allow for seamless database and component updates to occur while the coordinator navigates the roster.

Brooklyn Botanic Garden Tree Explorer

Explore Brooklyn Botanic Garden’s Living Collection Database by clicking through this expanding and collapsing D3 tree.

  • Used D3.js v5 to create a tree data-visualization giving users the ability to find a list of plant specimens in any part of the gardens.
  • Animated the collapse and expansion of tree branches using JS DOM manipulation to increase user’s interaction with the site.
  • Converted BBG’s csv database into hierarchical structure utilizing a combination of D3.js and custom algorithms.
